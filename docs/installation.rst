Installation
============

FLoRIN can be installed with all of its Python dependencies through the Python
Package Index or Anaconda.

PyPI
----

::

    pip install florin

Anaconda
--------

::

    conda install -c jeffkinnison florin

Python Dependencies
-------------------

* Python 3.4+
* numpy
* scipy
* scikit-image
* pathos
* mpi4py
* h5py
